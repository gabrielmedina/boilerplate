### Como utilizar

Você precisa ter instalado em sua máquina:
- NodeJS
  - Grunt
  - Bower


- Ruby
  - Sass

Caso possua os itens supracitados, basta baixar as dependências do NPM:

```shell
npm install
```

Baixar as dependências do Bower:

```shell
bower install
```

Ativar o Grunt:

```shell
grunt w
```

E ser feliz! :)
